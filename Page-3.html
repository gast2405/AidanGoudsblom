Robot nonsense

Robot nonsense is a GMTK game i made together with some friends in 5 days.

It features a robot entering a robot scaling experimentation laboratory.

Early version of the game.

The theme of GMTK 2024 was: build to scale.

The robot would solve puzzles troughout the laboratory by scaling boxes and eventually himself. The basic gist of the game is that you can't pickup boxes that are bigger then you, so to be able to pick them up you would have to get them trough shrink lasers. If a box is smaller then your size then you can also throw the box.

This was the our whiteboard on day 1 thinking of possible game ideas.

This was the first version of the game build in Unity.

We used Unity Collaborate to work on the same project together, i was the lead programmer with a friend helping me out.

The game gets more complex the further you progress, encountering new items and types of boxes. For example the bouncy box and the glass box.

The bouncy boxes introducing the shrink and grow mechanic.

The glass boxes slide over the ground and don't get affected by the lasers.

Around the end of the game the player also gets a laser gun that allows full control over growing and shrinking boxes. Some of the last puzzles include this mechanic before the game ends.

The laser gun allowing the player to both shoot grow and shrink lasers.

The end of the game where the player grows to an enormous size.
