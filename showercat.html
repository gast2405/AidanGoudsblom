Showercat

Showercat is a game where you help a kitten knock over the shampoo bottle in the bathroom, by masking layers so the cat can reach its goal.

Using an additive mask to solve the level.

Using a subtractive mask to solve the level.

The game was made in 48 hours for the Let's game Tokyo global game jam.

The theme of the game jam was masks, we were all brainstorming ideas for games revolving around this theme. The theme made me think of masks in photoshop and how you use them to only show the part of the layer that you want to show.

I thought this could be used in a platformer where you would have to mask parts of a layer to make platforms to reach the goal. When we all voted on game ideas we liked from the group, mine was chosen which i was very happy about :]
